# Cronix For Android Installation
---
To install cronix on Android, there are two ways:
---
### Method 1: Usb/Wireless Debugging (Needs: a computer with adb and git installed, dev options enabled on your phone and the two succesfully conntected through wireless or usb debugging)
First, open a terminal:
```txt
Windows Key + R -> cmd -> enter
```
Or (for linux):
```txt
CTRL + ALT + T
```
Then git clone the repo:
```bash
git clone https://github.com/james15-spec/Cronix-browser.git
cd Cronix-browser/android
```
After that, with adb installed, install the apk:
```bash
adb install cronix-browser.apk
```
### Method 2: Installing the app from the phone
As of Summer 2026, We are not on the play store but are hoping to be soon! Email us [Here](mailto:croftonixstudios@outlook.com) to get in touch!
Using a usb stick, cloud service or sd card, transfer the cronix-browser.apk to your phone or tablet and open it using a file manager. 
It may ask you to give the file manager permission to install apps, give the permission. It may also ask you to scan the app for viruses, if so, click scan app 
then once it is done, click install and it will install the app, Cronix Browser will now be installed!
